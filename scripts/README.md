# Скрипты администратора EduFlow

## replace_domain.py

Скрипт для замены домена `cysu.ru` на `cysu.ru` во всех файлах проекта.

### Использование:

```bash
# Базовая замена домена
python3 scripts/replace_domain.py

# Замена с очисткой лог-файлов
python3 scripts/replace_domain.py --clear-logs

# Включить лог-файлы в поиск и замену
python3 scripts/replace_domain.py --include-logs

# Все опции вместе
python3 scripts/replace_domain.py --clear-logs --include-logs
```

### Опции:

- `--clear-logs` - Очистить лог-файлы от старых записей с упоминаниями старого домена
- `--include-logs` - Включить лог-файлы в поиск и замену (по умолчанию пропускаются)

### Что делает скрипт:

1. 🔍 Находит все файлы с упоминаниями `cysu.ru`
2. 📄 Показывает список найденных файлов с номерами строк
3. 🔄 Выполняет замену на `cysu.ru`
4. 🧹 При необходимости очищает лог-файлы
5. 📊 Выводит статистику выполненных замен

### Пример вывода:

